## Виконання лабораторної роботи

Виконання лабораторної роботи передбачає вивчення прийомів роботи з системою контролю версій Git:

* створення віддаленого репозиторію;
* клонування віддаленого репозиторію;
* виконання комітів;
* створення гілок та перемикання між ними;
* автентифікація на GitHub за допомогою SSH;
* вивантаження локального репозиторію на GitHub.

## У практичній роботі №1 було

* створено локальний репозиторій;
* створено віддалений репозиторій на GitHub;
* налаштовано автентифікацію через SSH;
* створено проєкт за допомогою Vite;
* перевірено роботу `.gitignore`;
* реалізовано трекер завдань;
* реалізовано позначення виконаних завдань;
* завантажено проєкт на GitHub.
