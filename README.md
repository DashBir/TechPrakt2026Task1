## Опис лабораторної роботи

У цій лабораторній роботі було вивчено основні принципи роботи із системою контролю версій **Git** та сервісом **GitHub**.

Під час виконання роботи були опрацьовані такі можливості:

* створення локального репозиторію;
* створення віддаленого репозиторію;
* автентифікація через SSH;
* створення проєкту за допомогою **Vite**;
* використання файлу `.gitignore`;
* виконання комітів у стилі **Conventional Commits**;
* редагування файлу `README.md` із використанням Markdown;
* завантаження локального проєкту на GitHub.
